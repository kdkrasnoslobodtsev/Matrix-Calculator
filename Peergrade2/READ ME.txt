Калькулятор матриц

Программа попросит вас ввести размеры матрицы, а потом ввести построчно и саму матрицу
Если введенные пользователем данные окажутся некорректными, то ему будет предложенно ввести данные снова
Потом Вам будет предложено выбрать операцию, которую можно совершить с матрицей:
1) Найти след матрицы
Если матрица не квадратная, то будет выведено соответствующее сообщение
Если найти след возможно, то ответ будет выведен в консоль
2) Транспонирование матрицы
На экран будет выведена транспонированная матрица
3) Сложение матриц
Программа попросит ввести вторую матрицу, чтобы провести операцию сложения
Если введенные пользователем данные окажутся некорректными, то ему будет предложенно ввести данные снова
Если размеры матриц не совпадают, то будет выведено соответствующее соощение
Если найти сумму возможно, то на экран будет выведена матрица, являющаяся суммой двух матриц
4) Разность матриц
Программа попросит ввести вторую матрицу, чтобы провести операцию разности
Если введенные пользователем данные окажутся некорректными, то ему будет предложенно ввести данные снова
Если размеры матриц не совпадают, то будет выведено соответствующее соощение
Если найти сумму возможно, то на экран будет выведена матрица, являющаяся суммой двух матриц
5) Произведение матриц
Программа попросит ввести вторую матрицу, чтобы провести операцию разности
Если введенные пользователем данные окажутся некорректными, то ему будет предложенно ввести данные снова
Если при введенных пользователем размерах посчитаь произведение матриц будет невозможным, то выведется соответствующее сообщение
Если найти произведение возможно, то будет выведена матрица, являющаяся произведением двух матриц
6) Умножение матрицы на число
Программа попросит ввести значение, на которое следует умножить матрицу
Если введенное значение некорректно, будет возможность повторить ввод
На экран будет выведена матрица умноженная на число
7) Нахождение определителя матрицы
Если матрица не квадратная, то будет выведено соответствующее сообщение
Если посчитать определитель возможно то будет выведено его значение
8) Решение системы линейных уравнений
Данная реализация поддерживает решения систем уравнений, гарантированно имеющих одно решение. Матрица системы имеет вид n * (n+1) и не содержит нулей на главной диагонали. Иначе могут выводиться некорректные ответы

После завершения вычислений пользователю будет дана возможность поработать еще с уже введенной матрицей
Если он не захочет этого делать, то ему будет дана возможность ввести новую матрицу или выйти из программы
